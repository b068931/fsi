За визначенням, IDE (інтегроване середовище розробки) — це програмний додаток, який надає програмістам комплексні засоби для 
розробки програмного забезпечення. IDE зазвичай складається з редактора вихідного коду, інструментів автоматизації збірки та 
зневаджувача. fsi-visual-environment не є IDE, оскільки не надає всіх функцій, які зазвичай асоціюються з IDE. Тому замість 
цього використовується термін «візуальне середовище». Оскільки воно надає візуальний інтерфейс для деяких інструментів, які 
має FSI. 

Основні можливості fsi-visual-environment включають:
- Простий редактор коду разом із можливостями перегляду файлів.
- Певний рівень інтеграції з транслятором FSI (див. use-fsi-translator.txt для отримання додаткової інформації).
- Певний рівень інтеграції з медіатором FSI (див. use-fsi-mediator.txt для отримання додаткової інформації).
- Переклади англійською та українською мовами.
- Підтримка тем (світла та темна теми).
